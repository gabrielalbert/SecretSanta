# TaskGame Frontend

React 18 application for the TaskGame.

## Setup

1. Install Node.js (v18+)
2. Install dependencies:
   ```powershell
   npm install
   ```
3. Update `.env` file with API URL (`VITE_API_URL`)
4. Start development server:
   ```powershell
   npm run dev
   ```

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally
- `npm test` - Placeholder (configure Vitest or your preferred runner)

## Components

- `Login/Register` - Authentication
- `Dashboard` - Main navigation
- `ChrisMa` - Task creation and assignment
- `ChrisChild` - Task completion
- `TaskResults` - View completed tasks

## Features

- JWT authentication
- File upload support
- Responsive design
- Anonymous task viewing
